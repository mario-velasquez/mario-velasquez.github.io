Proudest Project

Project Start Spring 2020
Degree Audit System for the Computer Science Department
Worked with the Computer Science Undergraduate Program Coordinator to build tools for students.

Motivation

In short, The project was an effort to supply tools for advisement and degree checking for students. The current UTRGV system for degree progress is cumbersome and many bugs, especially since the university merged a few years causing course issues for students

Database Framework

The database was made using a gem known as Datamapper for Ruby. The database needed to hold many tables with attributes that pertain to courses and other features we intended to impliment. During development a basic SQL Server was hosted on our machine for our API and Frontends to have easy access

Backened API

Where is the project now?

Since I have always leaned towards the Computer Science Department, I was aproved to work with the Senior Design under the Computer Science Department despite being a Computer Engineer. The difference being I had to account for both departments requirements, and as a Computer Engineer I am required Senior Design I and II , whilst for a Computer Science Student it is only one Senior Design course. This project will be continued and I will lead another 3 senior computer scientist for the advancement of the project before I too myself will hand off the torch.
